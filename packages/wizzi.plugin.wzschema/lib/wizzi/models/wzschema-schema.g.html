Element Tag Super Flags
include include schemaitem
Description
Represent the root node of another external schema.   The wzName contains a blank separated name-value pair:   - the name is the type name of the external wzschema   - the value is the wzTag of the root node of the external wzschema (defaults to name)   see wizzi/models/bootstrap/t/wzschema/include.js.ittf   examples   include js module   - js : the external wzschema   - module : the wzTag of the root node of the js wzschema   include css   - css : the external wzschema   - css : the wzTag of the root node of the css wzschema   Children nodes will be loaded by the model factory of the external schema,   whose plugin must be listed in the plugin configuration when creating   the wizzi.wizziFactory instance.
