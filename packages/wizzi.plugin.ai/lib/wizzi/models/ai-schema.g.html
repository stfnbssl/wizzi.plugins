Defines the agent’s function within the crew. It determines the kind of tasks the agent is best suited for.

The individual objective that the agent aims to achieve. It guides the agent’s decision-making process.  
backstory backstory string
Provides context to the agent’s role and goal, enriching the interaction and collaboration dynamics.  
llm llm string
optional   Represents the language model that will run the agent. It dynamically fetches the model name from the OPENAI_MODEL_NAME environment variable, defaulting to “gpt-4” if not specified.

Specifies the language model that will handle the tool calling for this agent, overriding the crew function calling LLM if passed.   Default is None.  
max_iter max_iter integer 25
Max Iter is the maximum number of iterations the agent can perform before being forced to give its best answer.  
max_rpm max_rpm string
optional   Max RPM is the maximum number of requests per minute the agent can perform to avoid rate limits. It’s optional and can be left unspecified.   Default value is None.  
max_execution_time max_execution_time string
optional   Max Execution Time is the maximum execution time for an agent to execute a task. It’s optional and can be left unspecified, with a default value of None, meaning no max execution time.  
verbose verbose boolean false
optional   Setting this to True configures the internal logger to provide detailed execution logs, aiding in debugging and monitoring.  
allow_delegation allow_delegation boolean false
optional   Agents can delegate tasks or questions to one another, ensuring that each task is handled by the most suitable agent.   Default is False.  
step_callback step_callback string
optional   A function that is called after each step of the agent.   This can be used to log the agent’s actions or to perform other operations. It will overwrite the crew step_callback.  
cache cache boolean true
optional   Indicates if the agent should use a cache for tool usage.  
system_template system_template string
optional   Specifies the system format for the agent.   Default is None.  
prompt_template prompt_template string
optional   Specifies the prompt format for the agent.   Default is None.  
response_template response_template string
optional   Specifies the response format for the agent.   Default is None.  
allow_code_execution allow_code_execution boolean false
optional   Enable code execution for the agent.  
max_retry_limit max_retry_limit integer 2
optional   Maximum number of retries for an agent to execute a task when an error occurs.  
use_system_prompt use_system_prompt boolean true
optional   Adds the ability to not use system prompt (to support o1 models).  
respect_context_window respect_context_window boolean true
optional   Summary strategy to avoid overflowing the context window.  
code_execution_mode code_execution_mode string safe
optional   Determines the mode for code execution: ‘safe’ (using Docker) or ‘unsafe’ (direct execution on the host machine).

safe using Docker  
restrict direct execution on the host machine
